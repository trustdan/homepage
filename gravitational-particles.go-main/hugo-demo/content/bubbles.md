# Bubbles Effect

This page demonstrates the "bubbles" particle effect. The particles resemble floating bubbles with size and opacity animations.

## Features:
- Blue circular particles resembling bubbles
- Size and opacity animations
- Random movement
- Interactive hover effect that enlarges nearby particles
