# Custom Red Particles

This page demonstrates custom parameters, overriding the default configuration with red particles.

## Customization Options

You can customize the particles effect with parameters in the shortcode:

```
{{</* particles color="#ff0000" */>}}
{{</* particles preset="snow" color="#00ff00" */>}}
{{</* particles preset="bubbles" color="#0000ff" */>}}
```
