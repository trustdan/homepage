# Spacy Dots Effect

This page demonstrates the "spacydots" particle effect. The particles are connected with lines and react to cursor movement.

## Features:
- White dots connected by thin lines
- Moderate movement speed
- Interactive grabbing effect on hover
- Particles are added on click
