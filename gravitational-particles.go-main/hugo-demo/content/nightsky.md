# Night Sky Effect

This page demonstrates the "nightsky" particle effect. The particles resemble stars in a night sky with subtle twinkling and slight movement.

## Features:
- White circular particles resembling stars
- Twinkling effect through opacity animation
- Subtle connecting lines between nearby particles
- Random movement and positioning
